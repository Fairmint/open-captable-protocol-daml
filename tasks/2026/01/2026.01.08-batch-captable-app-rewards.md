# Task: Add App Rewards to Batch UpdateCapTable Choice

**Date**: 2026-01-08 **Status**: ✅ Completed **Target Repo**: `open-captable-protocol-daml`
**Related PR**:
[#94 - feat: add batch UpdateCapTable choice](https://github.com/Fairmint/open-captable-protocol-daml/pull/94)

## Goal

Extend the batch `UpdateCapTable` choice to accept featured app reward information, enabling the
caller to specify how many coupons should be created per batch call along with the FeaturedAppRight
contract ID.

## Summary of Changes

This task has been completed. The implementation differs slightly from the original proposal:

1. **Removed `featured_app_right` from Context entirely** - Instead of falling back to a
   context-stored value, the `UpdateCapTable` choice requires the caller to explicitly pass the
   `AppRewardsConfig` if they want markers created
2. **`appRewards = None` means no markers** - When `appRewards` is `None`, no markers are created at
   all (this is the expected behavior for individual OCP operations)
3. **Added `AppRewardsConfig` type** - Contains `couponCount` and `featuredAppRight` for explicit
   control
4. **Added tests** - Created `TestUpdateCapTable.daml` with tests for all scenarios

## Implemented Choice

```daml
-- | App reward configuration for batch operations
data AppRewardsConfig = AppRewardsConfig with
    couponCount: Int
    featuredAppRight: ContractId FeaturedAppRight
  deriving (Eq, Show)

-- | Batch update choice - the ONLY place where activity markers are created
nonconsuming choice UpdateCapTable : ()
  with
    appRewards: Optional AppRewardsConfig
  controller context.issuer
  do
    case appRewards of
      Some config -> do
        let beneficiaries = [AppRewardBeneficiary with beneficiary = context.system_operator, weight = 1.0]
        forA_ [1..config.couponCount] $ \_ -> do
          createActivityMarker beneficiaries config.featuredAppRight
      None -> pure ()
    pure ()
```

## API Usage Example

```daml
-- Without app rewards (no markers created)
submit issuer do
  exerciseCmd capTableCid UpdateCapTable with
    appRewards = None

-- With app rewards (creates couponCount markers)
submit issuer do
  exerciseCmd capTableCid UpdateCapTable with
    appRewards = Some AppRewardsConfig with
      couponCount = 5
      featuredAppRight = myFeaturedAppRightCid
```

# Task: Stateful Cap Table with OCF Object References

**Date**: 2026-01-05 **Status**: Completed ✅ **Target Repo**: `open-captable-protocol-daml`

## Goal

Implement ADR-002: Replace the event-sourcing pattern with a stateful `CapTable` contract that
maintains `Map Text ContractId` for all OCF objects, enabling O(1) lookups and reference validation
on create.

## References

- [ADR-002: Stateful Cap Table](https://github.com/Fairmint/open-captable-protocol-daml/blob/main/docs/adr/002-stateful-issuer-with-position-tracking.md)
- [Current Implementation](https://github.com/Fairmint/open-captable-protocol-daml/tree/main/OpenCapTable-v30)

## Key Changes

| Before                                  | After                                               |
| --------------------------------------- | --------------------------------------------------- |
| `Issuer` has ~45 factory choices        | `Issuer` is pure OCF data (no choices)              |
| Each OCF contract has `ArchiveByIssuer` | No `ArchiveByIssuer` (same signatories as CapTable) |
| No reference validation                 | O(1) validation that referenced objects exist       |
| Must replay events off-chain            | Maps show current state directly                    |

## CapTable Structure (from ADR)

```haskell
template CapTable
  with
    context: Context
    -- Issuer (exactly 1, edit only)
    issuer: ContractId Issuer
    issuer_id: Text
    -- Objects (Add/Edit/Delete)
    stakeholders: Map Text (ContractId Stakeholder)
    stock_classes: Map Text (ContractId StockClass)
    stock_plans: Map Text (ContractId StockPlan)
    stock_legend_templates: Map Text (ContractId StockLegendTemplate)
    vesting_terms: Map Text (ContractId VestingTerms)
    valuations: Map Text (ContractId Valuation)
    documents: Map Text (ContractId Document)
    -- Transactions (Add/Edit/Delete)
    -- Stock transactions
    stock_issuances: Map Text (ContractId StockIssuance)
    stock_transfers: Map Text (ContractId StockTransfer)
    stock_cancellations: Map Text (ContractId StockCancellation)
    -- ... (all other transaction maps)
```

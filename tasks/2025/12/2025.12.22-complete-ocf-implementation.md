## Implementation Pattern

For each new type:

1. Create DAML module in `OpenCapTable-v25/daml/Fairmint/OpenCapTable/`
2. Define data type matching OCF JSON schema (field order matters)
3. Add choice to `Issuer.daml` for creation
4. Add validators in `Types.daml` if new enums/types needed
5. Create test in `Test/` directory
6. Update `OCF_IMPLEMENTATION_STATUS.md`

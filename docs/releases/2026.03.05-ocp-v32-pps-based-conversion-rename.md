# Release: OCP v32 Major Upgrade

**Date**: 2026-03-05
**Networks**: devnet, mainnet

## Summary

This is a major version upgrade from OpenCapTable-v31 (0.0.4) to OpenCapTable-v32 (0.0.1). The change renames conversion mechanism constructors to align with OCF schema `PPS_BASED_CONVERSION` naming.

## What's Being Deployed

| Package | Version | Description |
|---------|---------|-------------|
| **OpenCapTable-v32** | 0.0.1 | PPS-based conversion constructor rename |

## Changes

- Renamed `OcfConversionMechanismSharePriceBasedConversion` ã `OcfConversionMechanismPpsBasedConversion` in `OcfConversionMechanism` enum
- Renamed `OcfConvMechSharePriceBased` ã `OcfConvMechPpsBased` in `OcfConvertibleConversionMechanism` sum type
- Renamed `OcfWarrantMechanismSharePriceBased` ã `OcfWarrantMechanismPpsBased` in `OcfWarrantConversionMechanism` sum type
- Data type `OcfSharePriceBasedConversionMechanism` unchanged (matches OCF schema file name)

## Breaking Changes

- All three constructor names changed (compile-time breakage for SDK consumers)
- Requires SDK converter updates to map new DAML names

## Deployment Commands

Run each step below (copy one command at a time):

```bash
# Step 0: Build all packages
npm run build

# Step 1: Upload OCP DAR
npm run upload-dar -- --package ocp --network devnet
npm run upload-dar -- --package ocp --network mainnet

# Step 2: Upload CouponMinter DAR
npm run upload-dar -- --package couponMinter --network devnet
npm run upload-dar -- --package couponMinter --network mainnet

# Step 3: Create OCP Factory (intellect as system_operator)
npx tsx scripts/create-ocp-factory.ts --network devnet
npx tsx scripts/create-ocp-factory.ts --network mainnet

# Step 4: Seed CouponMinter (intellect as operator)
npx tsx scripts/create-couponMinter.ts --network devnet
npx tsx scripts/create-couponMinter.ts --network mainnet

# Step 5: Backup DARs (after successful mainnet upload)
npm run backup-dar -- --package OpenCapTable-v26 --version 0.0.1 --network mainnet
npm run backup-dar -- --package CouponMinter --version 0.0.1 --network mainnet

# Step 6: Verify DAR backups
npm run verify-dars
```

## Notes

- CouponMinter uses consuming choices (`MintCoupons`), so the contract ID changes on every call. Clients must query for the current contract rather than using a stored contract ID.
- The `intellect` party is used as the operator for CouponMinter.
- Both `intellect` and `5n` providers receive the DAR uploads.
